# The Coffee Igloo

The Coffee Igloo es una aplicación web full-stack que ofrece recetas saludables de café helado a través de una API REST.

## Descripción del Proyecto

La aplicación permite a los usuarios:

* Crear nuevas recetas de café
* Visualizar todas las recetas
* Consultar una receta específica
* Actualizar recetas existentes
* Eliminar recetas

Este proyecto ha sido desarrollado como parte de un curso de programación y demuestra la implementación de una aplicación CRUD completa utilizando FastAPI y una base de datos relacional.
